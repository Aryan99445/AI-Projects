# AI Projects

A collection of AI and Machine Learning projects built with Python, covering computer vision, natural language processing, AI agents and DevOps automation.

<br>

## Projects:
<br>

**1. AI Multi-Agent Research Assistant:**

An automated research pipeline that takes a topic, searches the web, summarizes sources and generates a structured report — all using 4 specialized AI agents working together.

**Tech:** Groq (Llama 3), DuckDuckGo Search, Rich

**Concepts:** AI Agents, Task Orchestration, LLM Usage, Automation

<br>

**2. Safety Compliance Detection System:**

A real-time PPE detection system trained on construction site safety data. Detects helmets, safety vests and face masks from images, video files and webcam input. Generates compliance alerts based on what gear is detected.

**Tech:** YOLOv8, OpenCV, Ultralytics, Roboflow

**Concepts:** Computer Vision, Object Detection, Real-time Inference

<br>

**3. Sentiment Analysis — Traditional ML vs BERT:**

Sentiment analysis on the IMDB Movie Reviews dataset comparing three classical machine learning models against DistilBERT. Includes a Gradio dashboard for live predictions.

**Tech:** Scikit-learn, HuggingFace Transformers, Gradio

**Concepts:** NLP, TF-IDF, Text Classification, Transformers

<br>

**4. AI DevOps Log Analyzer (RAG):**

An AI-powered log analysis system that reads server logs, detects anomalies, identifies root causes and generates a structured incident report using a RAG pipeline.

**Tech:** Groq (Llama 3), ChromaDB, Sentence Transformers

**Concepts:** RAG, Vector Search, Log Analysis, DevOps Automation

<br>

## Tools and Languages

Python, PyTorch, Scikit-learn, HuggingFace, YOLOv8, ChromaDB, Groq, Gradio, OpenCV
